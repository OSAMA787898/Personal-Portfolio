Shortest-Path-finding-Between-Metro-Stations-Project-main

The Delhi Metro Route Finder is a Java-based program that helps users determine the shortest metro route and fare between two stations. It utilizes Graph and Heap data structures, where stations are represented as nodes and distances as edges. Algorithms like Dijkstra, BFS, and DFS are used to find the shortest path, and the fare is calculated based on the total distance traveled. The program includes a metro map for better navigation, making it a useful tool for commuters.
